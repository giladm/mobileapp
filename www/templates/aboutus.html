What's On My Plate?

What's On My Plate is a mobile app that gives the user nutrition information about a meal when the user snaps a picture of it.

How it Works?

The project combines mobile device technology with Watson Visual Recognition API to recognize the ingredients on a dish and report the nutrients information of that dish.

If the dish was not found the user will have the opportunity to train Watson with the new dish. At this point, the new dish will be available for the rest of the app community. This will insure continue learning of the platform.
